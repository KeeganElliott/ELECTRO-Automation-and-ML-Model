# ELECTRO DSP Analysis

This project applies Digital Signal Processing (DSP) techniques to ELECTRO Integrated simulation exports for high-voltage bushing analysis.

## Methods

- Fast Fourier Transform (FFT)
- Wavelet Transform Analysis

## Purpose

Investigate electric field and charge behavior during BIL impulse testing and determine whether frequency-domain or time-frequency techniques reveal failure-related patterns.

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
- SciPy
- PyWavelets
