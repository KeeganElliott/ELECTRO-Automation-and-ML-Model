# ELECTRO API & DSP Automation

## Overview

This project combines ELECTRO Integrated API automation with Digital Signal Processing techniques to automate the analysis, optimization, and evaluation of high-voltage bushing designs.

The repository began as a collection of FFT and Wavelet-based analysis tools for evaluating ELECTRO simulation exports and has since expanded into a broader automation framework for simulation setup, data extraction, and signal processing.

## Objectives

The primary objectives of this project are:

* Automate repetitive ELECTRO simulation tasks through the ELECTRO API
* Reduce engineering analysis time and manual data collection
* Extract simulation data for post-processing and evaluation
* Apply DSP techniques to identify trends, patterns, and failure indicators
* Create a scalable workflow for high-voltage insulation and bushing analysis

## Current Features

### DSP Analysis

* Fast Fourier Transform analysis
* Wavelet Transform analysis
* Automated signal visualization
* Frequency-domain investigation of simulation results

### ELECTRO API Automation

* Automated voltage assignment
* Automated object selection and material assignment
* Automated grounding and geometry modification
* Simulation workflow automation

## Future Development

Planned enhancements include:

* Automated batch simulation execution

* Spatial FFT analysis of electric field distributions

## Applications

This project is focused on high-voltage engineering applications, including:
